## 2. How did you use AI as a teammate?

- Which AI tools did you use on this project (for example: ChatGPT, Gemini, Copilot)?

I used Claude Code as my AI tool throughout this project.

## fixing the Hard/Normal range swap:
I identified that in the code, the ranges for Normal (1–50) and Hard (1–100) were swapped, meaning Hard was actually easier than Normal. It suggested swapping the return values so Normal maps to 1–50 and Hard maps to 1–100. I verified this by reading the fixed function and confirming that a higher upper bound means more possible values to guess from, which makes the game harder. I also ran the pytest regression test `test_hard_range_is_larger_than_normal`, which passed after the fix, confirming the ranges are now ordered correctly.

## fixing the Go Lower/Higher hints:
I identified that in the code, the response for a guess input of 1 was to "Go Lower" and for 100 to "GO Higer" despite the values being the lower and upper bounds of the game. The AI suggested to swithed the statements since the messages are backwards. The suggestion was correct. Additonally the AI noted that the check_guess fuction had not been impleted yet in logic_utils.py and deemed it necessary. So it imported the function from app.py. I verfied the result by launching the game.
